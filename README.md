# 🌿 Veloura Health
**Luxury Wellness Guided by Modern Pharmacy**

Veloura Health is a digital pharmacist assistant and premium wellness e-commerce experience. It combines pharmaceutical-grade guidance, elegant UI/UX, and modern wellness routines into a cohesive, portfolio-ready web application.

![Veloura Health Preview](https://via.placeholder.com/1200x600/F9F7F2/2C2C2C?text=Veloura+Health+Preview)

## ✨ Features
- 💊 **Digital Pharmacist Assistant** – Interactive chat UI with pharmacy-backed recommendations
- 🌙 **Symptom Support** – Expandable cards for sleep, stress, energy, immunity & focus
- 🌿 **Ingredient Education** – Clean, scientific breakdowns of core compounds
- 🕰️ **Wellness Routine Builder** – Circadian-aligned morning/evening protocols
- 🛍️ **Curated E-Commerce** – Shopify-inspired product cards with soft hover states
- 📧 **Premium Newsletter** – Glassmorphic subscription section
- 📱 **Mobile-First Responsive** – Optimized for all screen sizes
- 🎨 **Luxury Aesthetic** – Ivory, sage, champagne palette with glassmorphism & soft shadows
